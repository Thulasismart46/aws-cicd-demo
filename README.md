# AWS CI/CD Artifact Storage Using S3 ☁️⚙️

This project demonstrates a beginner-friendly CI/CD pipeline using:

* AWS CodePipeline
* AWS CodeBuild
* Amazon S3
* GitHub Integration

## Project Workflow

GitHub Repository → CodePipeline → CodeBuild → S3 Artifact Storage

## Services Used

* Amazon S3
* AWS CodePipeline
* AWS CodeBuild
* IAM
* GitHub

## Features

* Automated build pipeline
* Artifact storage in S3
* GitHub integration
* Continuous Integration workflow
